#+title: Math Stake

* 目的

这是一个网页应用, 帮助小学生练习数学, 目前主要是一年级的口算, 后续会逐步迭代加入其它内容.

使用场景:

- 部署在家庭内的局域网, 只需要考虑非常少的用户
- 小学生通过 ipad 或手机浏览器打开网站, 选择要做的测试
- 网站出题, 等待小学生填入答案. 同时开始倒计时, 倒计时结束后禁止继续答题
- 倒计时结束前, 小学生可以主动点击按钮表示"交卷"以结束答题
- 答题结束后, 网站给出评分, 并保存结果 (所有的题目, 答案, 答题总时间)
- 管理员可以随时查看所有练习结果

注意:

- 用 python django 完成, 管理员直接用 django admin 界面做管理
- 除此之外, 不需要用户认证等功能, 用户可以直接打开除 admin 页面以外的内容开始使用
  - 小学生无需登陆或输入名字即可使用网站. 保存的测试结果也是匿名的, 只要记录生成时间即可
- 倒计时在前端实现, 不用假设小学生会 hack 网站

* 导航结构

页面划分:

- 首页: 呈现可选的多个测试, 小学生可以点击某个测试进入答题
- 试卷页: 呈现多个题目, 等待小学生填入答案. 目前只需支持计算题, 小学生在答题框中填入数字. 答题页上有浮动的倒计时, 确保在网页可见范围内出现, 提醒小朋友还剩下的时间
- 结果页: 给出评分和时间, 列出有错的题目

跳转关系:

首页 -> 试卷页 -> 结果页 -> 首页

答题页 -> 首页

* 视觉风格

采用简洁素雅的视觉风格

* 技术约束

- 用 python django 完成
- 使用本地的 sqlite 数据库
- django server 直接服务, 不需要考虑 load balancer 之类的技术

* 计算类型 A: 加减法

注意: 同一个计算类型可以通过不同的参数配置形成多个不同的测试 (即呈现在首页上供挑选的测试)

该计算类型由计算机自动生成题目和验证答案. 注意:

- 每个题目是一个包含 N 个数字(不超过 X 的自然数, 可以有 0)和 N-1 个算符的加减法问题
  - 没有括号
  - 算符只包括加法和减法
  - 计算结果必须是一个不小于 0 的自然数
- 如果有多个算符, 参数 P 控制从左到右的计算过程中是否允许出现负数
  - P == True 表示允许, 否则表示不允许
- 每个试卷包含 M 个不重复的题目, 并且要求在 T 秒内完成
  - 如果无法生成 M 个, 尝试一定次数后有多少个就用多少个
- 需要保证每道题目的计算过程中有进位或退位
  - 如果有多个算符, 按从左到右的计算过程中必须保证 *每个计算* 都有进位或退位
    - 例如, 9 + 8 - 7 的第二步没有退位或进位, 不符合要求
    - 例如, 8 + 8 - 7 的两步都有进位或退位, 符合要求
    - 对多位数, 在任何位置上发生进位或退位即可
- 管理员可以设定 N, X, M, P, T
